¿Qué cosa es Karel?

Karel el robot es un pequeño proyecto y una obsesión. Sin duda nació de un delirio de grandeza, y ahora está aquí, para ser útil en tu formación como programador, o quizá para entretenerte un rato con problemas interesantes.

Se trata de un lenguaje de programación orientado al aprendizaje de la programación, con una sintaxis mínima y sencilla, fácil de aprender, e instrucciones básicas de lo más reducidas, con las cuales se pueden resolver problemas monstruosos.

Usando karel puedes aprender técnicas, mejorarlas o ponerte retos bastante interesantes, creo que basta con echarle un ojo a la lista de problemas para darse cuenta de eso.

El proyecto

Este proyecto pretende motivar el uso de Karel como plataforma para el aprendizaje de la programación, o simplemente para retar tus conocimientos de las estructuras de control y otras cosas.
